### 说明 

    models --------- file  文件处理
    views ---------- 模板引擎 ejs 视图页面
    contorller ----- router 路由
    public --------- css、js 静态文件
    uploads -------- 图片相册

    app.js --------- 入口


    mvc 模型  models(模组)  views(视图)  contorller(控制)

### 启动


    1、下载代码库 git clone ...
    2、进入项目文件夹 npm install 
    3、在项目文件夹 node app.js
    4、浏览器打开 http://127.0.0.1:3000
